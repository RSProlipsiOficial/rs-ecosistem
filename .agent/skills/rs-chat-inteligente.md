  Skill que define como construir chats inteligentes
  com memória, contexto, ações e governança.
---

## OBJETIVO
Criar chats que fazem, não só respondem.

## CAPACIDADES
- memória de conversa
- contexto por usuário
- execução de ações
- integração com dados reais

## REGRAS
- Chat sem contexto é inútil
- Ação sem permissão é bloqueada
- Toda ação é auditável
