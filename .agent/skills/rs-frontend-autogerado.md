## OBJETIVO
Garantir UI profissional e consistente em qualquer produto RS.

## COMPONENTES OBRIGATÓRIOS
- Sidebar
- Topbar
- Dashboard
- CRUD padrão
- Estados: loading, vazio, erro
- Responsivo (desktop/mobile)

## PADRÃO VISUAL
- Dark + Gold (RS)
- Tipografia limpa
- Cards e grids consistentes

## REGRAS
- UI feia é bug
- CRUD sem estado vazio não entrega
