## OBJETIVO
Publicar landing page profissional com controle total.

## PASSOS
1) Definir objetivo da landing
2) Criar layout no padrão RS
3) Integrar formulário/CTA
4) Criar admin de edição
5) Integrar métricas e SEO
6) Testar responsividade
7) Publicar

## SAÍDA
- Landing page publicada
- Admin funcional
