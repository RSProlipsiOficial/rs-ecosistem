## OBJETIVO
Criar um chat que executa ações reais via conversa.

## PASSOS
1) Definir propósito do chat
2) Aplicar rs-chat-inteligente
3) Definir memória e contexto
4) Integrar fontes de dados
5) Definir ações permitidas
6) Implementar auditoria
7) Testar cenários críticos

## SAÍDA
- Chat funcional
- Contexto persistente
- Ações governadas
